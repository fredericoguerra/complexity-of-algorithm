# Insertion Sort Algorithm
This repo has the Insertion Sort algorithm implementation in Python 3.9.0.

## 1. Run the main file
To run this project locally after pushing into your local repository, run this commands below:

and
```
python main.py
```

Once you run the Insert Sort script, you'll see the menu below containing one example on top of it. You must type 1, 2 or 3 to test in 4 different arrays of 1000, 10000 and 100000 elements, respectivelly.

### 2. First Menu - Algorithm selection:

```
-----------------------------------------------------------------------
Please select which algorithm to test
1 -- Insertion Sort
2 -- Selection Sort
3 -- Quick Sort
4 -- Merge Sort
5 -- Exit
Enter your choice:
```
